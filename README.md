#  Consulta de Clima en Tiempo Real

¡Bienvenido a mi proyecto de consulta climática! Esta aplicación web permite a los usuarios obtener información meteorológica precisa de cualquier ciudad del mundo, con un diseño moderno y una experiencia de usuario optimizada.

---

##  Sobre el Proyecto

Este proyecto nació como parte de mi formación como **Desarrollador Full Stack**. El objetivo principal fue crear una herramienta funcional que consuma datos de APIs externas y los presente de forma amigable.

### Características principales:
* **Búsqueda Dinámica:** Localiza ciudades mediante geocodificación.
* **Datos en Tiempo Real:** Muestra temperatura exacta y estado del cielo.
* **Interfaz Adaptable:** Diseño responsivo con gradientes modernos.
* **Traducción Inteligente:** Convierte códigos técnicos de la API en descripciones con emojis (ej. 🌧️, ☀️, ☁️).

### Tecnologías utilizadas:
* **HTML5 & CSS3:** Estructura y diseño (Flexbox, Gradients).
* **JavaScript (ES6+):** Lógica asíncrona y manipulación del DOM.
* **Open-Meteo API:** Servicio gratuito para datos meteorológicos y geocodificación.
* **Git & GitHub:** Control de versiones y despliegue.
